# DBeaver SQL 脚本

本目录是 [使用说明.md](../../使用说明.md#sql-整理语句) 与 [README.md](../../README.md) 中**已测试 SQL** 的脚本化整理，便于在 DBeaver 中直接打开。  
**不替代**文档内原文——文档保留完整语句，此处按功能拆分、写入操作加醒目标注。

## 命名规则

| 前缀 | 含义 |
|------|------|
| `查询-` | **只读**，不修改数据 |
| `写入-` | **会改库**（`UPDATE` / `DELETE`），文件头与写入段有 ⚠️ 标注 |

写入脚本结构固定为两段：

1. **预览** — `SELECT`，先看将影响哪些行  
2. **写入** — `UPDATE` / `DELETE`，确认后再单独选中执行

## 使用注意

- 写入前建议备份 `data/contacts.sqlite`，或在 DBeaver 中先 **导出** 相关表。
- 写入脚本中的 `${ids}`、`${contact_ids}` 为 DBeaver 绑定变量；不用变量时直接改成具体 id 列表。
- 删整人优先用 [写入-删除联系人.sql](写入-删除联系人.sql)；也可用 Python `6-del_contact.py`。
- 合并重复联系人用 `7-merge_contact.py`，不要手写 SQL 合并。
